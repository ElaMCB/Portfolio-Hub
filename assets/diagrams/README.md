# Architecture Diagrams

This directory contains architecture diagrams and system design visuals.

## Contents

- System architecture diagrams
- Data flow diagrams
- Component interaction diagrams
- Deployment diagrams

## Formats

Diagrams are stored in:
- Mermaid format (`.mmd`) - For version control
- PNG/SVG format - For documentation and presentations

---

*Diagrams are updated as architectures evolve.*
