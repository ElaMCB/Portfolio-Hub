# Project Logos

This directory contains logos and branding assets for ElaMCB projects.

## Usage

Logos can be used in:
- README files
- Documentation
- Presentations
- Portfolio websites

## File Naming Convention

- `project-name-logo.png` - Main logo
- `project-name-icon.png` - Icon version
- `project-name-banner.png` - Banner/header version
