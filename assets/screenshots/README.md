# Project Screenshots

This directory contains screenshots and visual demonstrations of projects.

## Organization

Screenshots are organized by project:
- `project-name/` - Project-specific screenshots
  - `ui/` - User interface screenshots
  - `features/` - Feature demonstrations
  - `demos/` - Demo screenshots

## Usage

Screenshots are used in:
- README files
- Documentation
- Portfolio websites
- Presentations
